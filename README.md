# 🏥🌐 Hospital Network Design — Cisco Packet Tracer

A complete **hospital network design and simulation** built using **Cisco Packet Tracer**, developed as a 3rd semester Computer Networks (CN) project at Air University.

The project simulates a fully functional hospital network infrastructure with multiple departments, secure communication, and proper IP addressing.

---

## 🏗️ Network Design Features

- 🏢 **Multi-department Layout** — Separate network segments for each hospital department
- 🔀 **Routers & Switches** — Full routing and switching configuration
- 📡 **Wireless Access Points** — WiFi connectivity for mobile devices
- 🔒 **Network Security** — Access control and secure communication setup
- 🌐 **IP Addressing** — Proper subnetting and IP allocation scheme
- 🖥️ **Server Configuration** — DNS, DHCP, and web server setup
- 📊 **Network Topology** — Star and hierarchical topology implementation

---

## 🏥 Hospital Departments Covered

- 🚑 Emergency Department
- 🧪 Laboratory
- 🏨 Patient Wards
- 👨‍⚕️ Doctor's Office
- 🏛️ Administration
- 💊 Pharmacy

---

## 🧠 Networking Concepts Used

- **Subnetting** — Dividing IP address space efficiently
- **VLAN Configuration** — Isolating department traffic
- **Routing Protocols** — Inter-department communication
- **DHCP** — Automatic IP assignment
- **DNS** — Domain name resolution
- **Network Security** — Firewall and access control basics

---

## 🛠️ Tools Used

- **Cisco Packet Tracer** — Network simulation
- **Networking Protocols:** TCP/IP, DHCP, DNS, HTTP

4. Explore the network topology and configurations
